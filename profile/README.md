# 🏥 Alô Doctor

Bem-vindo ao ecossistema **Alô Doctor**, uma solução tecnológica dedicada à gestão e modernização do atendimento na área da saúde.

## 📋 Sobre o Projeto

O Alô Doctor foi concebido para centralizar processos clínicos, facilitar a comunicação entre profissionais de saúde e pacientes, e garantir a integridade dos dados médicos através de uma arquitetura robusta e escalável.

## 🏗️ Arquitetura do Sistema

O sistema está dividido em dois pilares principais para garantir a separação de responsabilidades e facilitar a manutenção:

* **[Web App](./web-alodoctor):** Interface intuitiva desenvolvida para profissionais e utilizadores finais.
* **[API REST](./api-alodoctor):** O motor do sistema, responsável pela lógica de negócio, autenticação e persistência de dados.



## 🛠️ Tecnologias Principais

* **Linguagem:** TypeScript
* **Frontend:** Next.js / React
* **Backend:** Node.js / Fastify
* **Base de Dados:** Prisma / PostgreSQL
